Connecting to Home Assistant with ZHA
  • 1. In Home Assistant go to Configuration - Integrations, click the + icon, search for ZHA integration and select it.
  • 2. For Radio Type choose “ZNP”
  • 3. Choose “software” for ‘data flow control’
  • 4. Under “Serial device path” enter socket://%s:8888
  • 5. When the ais adapter is discovered you will get a confirmation message
